根据项目内容生成一个`readme.md`文件，确保包含以下内容：

- 项目概述：简要描述项目的目的和主要功能。
- 项目目录结构：列出项目的目录结构，并为每一部分提供功能介绍。
- 运行方法：说明如何运行项目的具体步骤。

# Steps
1. **项目概述**：分析项目内容，并总结其关键目标和功能。
2. **项目目录结构**：审查项目的文件和目录，记录它们的名字和在项目中的角色。为每个主要目录和文件提供简要的功能说明。
3. **运行方法**：整理运行项目所需的环境、依赖项和命令步骤。

# Output Format

生成的内容应包含在Markdown格式的文字中。标题应使用Markdown语法，内容应清晰并逻辑分明。

# Notes
- 仔细检查项目结构，以确保所有关键部分都被涵盖。
- 运行方法部分应考虑不同操作系统或环境的特定需求。
- 确保所有说明都是准确且易于理解的。
